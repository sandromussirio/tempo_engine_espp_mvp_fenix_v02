---

## 1. Identidade do projeto

Este repositório contém o MVP local do **Tempo Engine / ESPP**.

O ESPP é um protocolo para transformar dados espaciais 2D estruturados em instruções geométricas executáveis.

O Tempo Engine é o motor que executa essas instruções.

O Blender é apenas o ambiente inicial de validação.

Este projeto **não é**:

- plugin de Blender;
- API;
- SaaS;
- interface web;
- sistema BIM;
- produto final.

Neste estágio, é um MVP técnico local.
